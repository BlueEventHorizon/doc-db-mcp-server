# NFR-003 標準ライブラリ優先 要件定義書

## 概要

検索インデックスの実装において Python 標準ライブラリを優先するが、外部ライブラリの使用も許容する。

> **注記**: REQ-001 NFR-01-1 では「Python 標準ライブラリのみ」が原則だが、本要件はセマンティック検索の精度・速度要件を達成するため、ベストエフォートに緩和する。この緩和はプロジェクトオーナーが承認済み。

## 前提条件

- なし

## 要件一覧

### ライブラリ選定方針

- Python 標準ライブラリ（sqlite3、json 等）で実現可能な場合はそちらを優先する
- 標準ライブラリでは精度・速度の要件を満たせない場合、外部ライブラリの使用を許容する
- 外部ライブラリを採用する場合、依存の追加理由と代替案の検討結果を設計書に記録する

### 適用範囲

- 検索インデックスの構築
- 検索の実行
- 検索結果のランキング

### エラーケース

| 条件                           | 動作                                         |
| ------------------------------ | -------------------------------------------- |
| 外部ライブラリが未インストール | エラーメッセージでインストール方法を案内する |
