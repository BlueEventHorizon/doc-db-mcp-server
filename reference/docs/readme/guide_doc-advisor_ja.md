# doc-advisor 詳細ガイド

AI 検索可能なドキュメントインデックス（ToC）生成・検索ツール。文書から AI がメタデータを抽出し、タスクに関連する rules/specs 文書を自動発見する。

## スキル詳細

### query-rules

```
/doc-advisor:query-rules [--toc|--index] タスクの説明
```

| 引数           | 説明                                     |
| -------------- | ---------------------------------------- |
| `--toc`        | ToC（キーワード）検索のみ                |
| `--index`      | Embedding（セマンティック）検索のみ      |
| （なし）       | ハイブリッド自動選択（デフォルト）       |
| `タスクの説明` | 関連するルール文書を検索するタスクの説明 |

ToC キーワード検索と Embedding セマンティック検索を組み合わせて、タスクに関連するルール文書（コーディング規約、アーキテクチャルール、ワークフローガイド等）を特定する。デフォルトのハイブリッドモードでは両方の検索結果を統合する。

### query-specs

```
/doc-advisor:query-specs [--toc|--index] タスクの説明
```

| 引数           | 説明                                   |
| -------------- | -------------------------------------- |
| `--toc`        | ToC（キーワード）検索のみ              |
| `--index`      | Embedding（セマンティック）検索のみ    |
| （なし）       | ハイブリッド自動選択（デフォルト）     |
| `タスクの説明` | 関連する仕様文書を検索するタスクの説明 |

ToC キーワード検索と Embedding セマンティック検索を組み合わせて、タスクに関連する仕様文書（要件定義書、設計書等）を特定する。

### create-rules-toc

```
/doc-advisor:create-rules-toc [--full]
```

| 引数     | 説明                                           |
| -------- | ---------------------------------------------- |
| （なし） | 差分更新（ハッシュベース）または中断からの再開 |
| `--full` | 全ファイルスキャン（初回作成または再生成用）   |

ルール文書の追加・変更・削除後に ToC を更新する。

### create-specs-toc

```
/doc-advisor:create-specs-toc [--full]
```

| 引数     | 説明                                           |
| -------- | ---------------------------------------------- |
| （なし） | 差分更新（ハッシュベース）または中断からの再開 |

仕様文書の追加・変更・削除後に ToC を更新する。
